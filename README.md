# Smart Fall Detection and Alert System using STM32

## Overview
This project is designed to detect abnormal body movement and falls using MPU6050 sensor and STM32 microcontroller.

## Components Used
- STM32 Nucleo F446RE
- MPU6050 Sensor
- 16x2 I2C LCD
- LED
- Buzzer
- PCB / Perfboard

## Features
- Real-time acceleration monitoring
- Fall detection using threshold values
- LCD alert display
- LED and buzzer emergency alerts
- I2C communication

## Connections
- PB8 → SCL
- PB9 → SDA
- PB0 → LED
- PA6 → Buzzer

## Software Used
- STM32CubeIDE
- STM32CubeMX
- Embedded C
- HAL Drivers

## Applications
- Elderly healthcare monitoring
- Hospitals
- Rehabilitation centres
- Smart healthcare systems
